# YouTube Playlist Time Calculator

A web application that calculates the total number of videos and total watch time of a YouTube playlist.  
It also estimates adjusted watch time based on playback speed and the number of days required to complete the playlist.

## Live Demo
https://ashipa-creek.github.io/youtube-playlist-calculator/

## Features
- Calculates total videos and playlist duration
- Supports playback speeds (1x, 1.25x, 1.5x, 2x)
- Estimates days required based on daily watch hours
- Light and Dark mode toggle for better user experience
- Uses YouTube Data API v3
- Deployed on GitHub Pages
